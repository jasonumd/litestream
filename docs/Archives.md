# Classical Music Archives

## classicalarchives.com

<https://www.classicalarchives.com/>

The current litestream project points to classicalarchives.com, which is a very complete archive with rich metadata and multiple high-quality recordings for just about every piece of music out there. I don't think a better archive exists.

## openopus.org

<https://openopus.org/>

This archive contains metadata about classical music, not recordings. Although it is not useful for litestream, it is interesting to know about.

## musopen.org

<https://musopen.org/music/>

This archive contains recording and is free. I can't see if it is well-organized and they don't have a public API.
